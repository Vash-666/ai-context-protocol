## 🎬 **Script Creation Process** (Stage 1)

### **Input:**
- Topic/achievement to document
- Technical details and context
- Target platform(s)
- Desired length/format

### **Process:**
1. **Analyze topic** (complexity, audience fit)
2. **Create STAR outline** (Situation, Task, Action, Result)
3. **Write script** (clear, concise, factual)
4. **Platform optimization** (adapt format for target)
5. **Quality check** (truth density, no hype)

### **Output Format:**
```markdown
# [Topic] - Teaching Script

## STAR Outline
- **Situation:** [Context]
- **Task:** [Challenge]
- **Action:** [Solution]
- **Result:** [Outcome with metrics]

## Full Script
[Detailed content...]

## Platform Adaptations
- GitHub: [Showcase format]
- LinkedIn: [Article format]
- X/Twitter: [Thread format]
```

### **Quality Standards:**
- Maximum truth density (cut fluff)
- No hype language (breakthrough, amazing, victory)
- Evidence-based (metrics, code, results)
- Recruiter-friendly (clear value proposition)
- Technical accuracy (validate with source material)

## 📱 **Social Media Transformation** (Stage 2)

### **Input:**
- Script from Stage 1
- Target platform(s)
- Engagement goals

### **Process:**
1. **Extract key points** from script
2. **Adapt format** for platform
3. **Optimize for engagement** (hooks, CTAs)
4. **Brand consistency check** (voice, guidelines)
5. **Final polish** (grammar, formatting)

### **Platform Specifications:**

#### **GitHub Showcase (PRIMARY)**
- **Format:** README.md with technical depth
- **Length:** 600-900 words
- **Structure:** Problem → Solution → Results → Code/Docs
- **Focus:** Recruiter demo value + technical credibility
- **Quality threshold:** ≥9.3/10 required

#### **LinkedIn Article**
- **Format:** Professional narrative
- **Length:** 400-800 words
- **Hook:** First 2 sentences critical
- **Structure:** Story → Lesson → Call-to-action
- **Focus:** Career value + learning outcomes

#### **X/Twitter Thread**
- **Format:** Multi-tweet thread
- **Length:** 5-10 tweets, 280 chars each
- **Hook:** Tweet 1 grabs attention
- **Structure:** Hook → Context → Solution → Results → CTA
- **Focus:** Quick wins + technical insights

#### **Instagram Carousel**
- **Format:** 5-10 slides
- **Length:** 50-100 words per slide
- **Visual:** Clean, minimal text
- **Structure:** Hook → Steps → Result → CTA
- **Focus:** Visual storytelling + key takeaways

#### **WhatsApp Broadcast**
- **Format:** Short message
- **Length:** 100-200 words
- **Tone:** Direct, friendly
- **Structure:** News → Link → Why it matters
- **Focus:** Immediate value + action
